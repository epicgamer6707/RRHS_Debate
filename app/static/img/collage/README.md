# Landing-page collage photos

Upload your team photos here, named exactly:

    1.jpg  2.jpg  3.jpg  4.jpg   ← scroll UP in the left column
    5.jpg  6.jpg  7.jpg  8.jpg   ← scroll DOWN in the right column

- Use `.jpg` (lowercase). JPGs keep the page light.
- Portrait-ish photos look best (tiles are ~2:3 tall).
- Until a file exists, its tile shows a dark placeholder.

To add them on GitHub: open this folder in the repo →
`app/static/img/collage/` → **Add file → Upload files** → drop the 8 photos →
commit. They appear on the landing page automatically.
